# 制单 Skill

## 用途
- 使用项目现有制单流程脚本自动完成填单与关闭确认。
- 直接复用同目录 `制单.py` 与项目公共模块 `ubank_common.py`。

## 执行步骤
1. 确认在项目根目录运行。
2. 执行 `python "skills/制单skill/制单.py"`。
3. 观察终端输出，确认已进入单笔转账经办并完成两次确认。

## 关键文件
- `制单.py`：制单主脚本（本 Skill 内置副本）
- `ubank_common.py`：登录、按键、点击等公共能力（项目根目录）
