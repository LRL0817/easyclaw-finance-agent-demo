# 批量制单 Skill

## 用途
- 使用项目现有流程脚本自动完成登录并进入批量转账经办页面。
- 直接复用同目录 `制单.py` 与项目公共模块 `ubank_common.py`。

## 执行步骤
1. 确认在项目根目录运行。
2. 执行 `python "skills/制单批量转账skill/制单.py"`。
3. 观察终端输出，确认已进入批量转账经办页面。

## 关键文件
- `制单.py`：批量制单主脚本（本 Skill 内置副本）
- `ubank_common.py`：登录、按键、点击等公共能力（项目根目录）
