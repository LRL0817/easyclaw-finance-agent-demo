# 兴业银行单笔转账脚本（1920x1080 / 100% 缩放）

## 目标环境

- 屏幕分辨率：1920x1080
- Windows 显示缩放：100%
- 桌面存在快捷方式：`兴业银行企业网银.lnk`
- Python 已安装，并能在命令行使用 `python`

## 第一次使用

1. 安装依赖：

```bat
pip install -r requirements.txt
```

2. 复制 `.env.example`，改名为 `.env`，并填写：

```text
LOGIN_PWD=网盾密码
CERT_PWD=登录页密码
```

3. 双击运行：

```text
run_单笔转账.bat
```

## 坐标调整说明

这个导出版从当前电脑的 125% 缩放坐标换算到目标电脑 100% 缩放，默认参数在 `open_bank.py` 顶部：

```python
SOURCE_DPI_SCALE = 1.25
TARGET_DPI_SCALE = 1.0
```

脚本会优先用 UIA 控件名点击“下一步”“提交”等按钮；只有控件名找不到时，才使用换算后的固定兜底点。
