# 脱敏版财务自动化项目代码

本目录是参赛提交用的公开安全版代码副本，来自本机财务付款自动化项目。

## 包含内容

- OA 合同付款数据获取流水线
- 合同付款新增监控脚本
- 招行 U-BANK 制单自动化脚本
- 工商银行 / 中国银行 / 农业银行 / 兴业银行入口脚本
- 各模块 SKILL.md 与说明文档

## 已移除内容

- `.env` 登录凭据
- 浏览器登录态 `.browser_profile`
- 真实付款 JSON、`bank_form.json`、`latest.json`、`state.json`
- 银行页面截图、日志、缓存、pyc 文件
- 二进制驱动、压缩包、PDF、DLL、EXE

## 已替换内容

- 真实 OA 域名替换为 `oa.example.local`
- 真实付款单位替换为示例付款单位
- 12 位以上连续数字替换为 `000000000000000000`
- 本机用户路径替换为 `%USERPROFILE%`

该目录用于评审理解项目结构和实现方式，不包含可直接连接真实银行或 OA 的敏感配置。
