# 中国银行单步支付

本 skill 用于通过 Python 控制的浏览器（Playwright）打开中国银行官网，并准备进行单步支付操作。

## 使用步骤

1. **检查依赖**：确认本机已安装 `playwright`，以及对应的 Chromium 浏览器内核。
   - 检查命令：`python -c "import playwright"`
   - 若未安装：`pip install playwright && python -m playwright install chromium`

2. **运行入口脚本**（仅弹浏览器，**不显示控制台窗口**）：
   - 推荐：双击工作目录下的 `启动.bat`。
   - 或手动执行（无控制台版 Python）：
     ```bat
     start "" pythonw open_boc.py
     ```
   - 如需查看脚本日志（会有黑色控制台窗口）则改用：
     ```bash
     python open_boc.py
     ```

3. **脚本行为**：
   - 以**有头模式**（headful，可见窗口）启动 Chromium。
   - 打开 `https://www.boc.cn/`。
   - 自动导航到企业网银登录页，完成 U盾登录。
   - 进入「付款业务」→「单步支付」页面。
   - 自动填入 `.env` 中配置的付款数据（收款人、账号、金额等）。
   - 用户**关闭浏览器窗口**（或终端按 Ctrl+C）后脚本退出。

4. **后续支付操作**：
   - 登录个人/企业网银。
   - 进入「转账汇款」→「单笔转账」/「单步支付」。
   - 填写收款账号、户名、金额、附言等。
   - 输入 U 盾/口令卡验证码完成支付。

## 注意事项

- 涉及真实资金操作，**禁止由脚本自动提交支付**。脚本仅负责打开浏览器、导航与填表，金额、收款人、验证码等关键操作必须由用户人工完成并核对。
- 如需进一步自动化（如自动填写收款人信息），需要用户明确授权，并在脚本中加入二次确认环节。
- 中行网银对浏览器环境与安全控件有较强要求，部分功能（如 U 盾签名）在 Playwright 内核中可能无法直接使用，必要时改用系统默认浏览器。
